
## 산기대생을 위한 인공지능 스피커
### AI Speaker for KPU Student

### Function
  - Snowboy를 이용한 음성호출
  - STT(Speech To Text), TTS(Text To Speech)
  - 웹 크롤링을 이용한 공지사항 알림
  - GPS 기반 날씨 알림
  - 셔틀버스 시간 알림
  - GUI
  
### Hardware
  - Raspberry PI 3 Model B
  - Raspberry PI Public 7' Touch Screen
  - USB Microphone
  
### Software
  - Python3
  - MariaDB
  - PyQt5
  
### APIs
  - Google Cloud Speech API
  - Naver TTS API
  - SK planet Weather API
